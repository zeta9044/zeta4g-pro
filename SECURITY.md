# Security Policy

## Supported Versions

Security reports are accepted for the latest published Zeta4G Pro release.
Older releases may receive fixes at Zeta4Lab's discretion.

## Reporting a Vulnerability

Do not open a public GitHub issue for a suspected vulnerability.

Report security issues by email:

- zeta4lab@gmail.com

Include the affected version, platform, deployment method, reproduction steps,
expected impact, and any relevant logs or proof-of-concept details. Zeta4Lab
will review the report and coordinate the response directly.

## Scope

In scope:

- Zeta4G Pro release binaries
- Official Docker Hub image `zeta4lab/zeta4g-pro`
- Official GHCR image `ghcr.io/zeta9044/zeta4g-pro`
- Release packaging and Docker runtime files in this repository

Out of scope:

- Unofficial images or redistributed binaries
- Modified deployments not produced by Zeta4Lab
- Denial-of-service testing against public infrastructure without permission
